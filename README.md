# 🚀 QR Master - Gerador de QR Codes e Códigos de Barras

Site: https://max-qr-code.netlify.app


> Um gerador de códigos moderno, rápido e com design premium. Crie QR Codes e Códigos de Barras personalizados, individualmente ou em massa.

## ✨ Funcionalidades

- **🎨 Design Premium:** Interface moderna com efeito Glassmorphism, animações fluidas e tema Escuro/Claro.
- **⚡ Geração em Tempo Real:** Digite e veja o código aparecer instantaneamente.
- **📦 Geração em Massa:** Crie até 800 códigos de uma vez (ideal para etiquetas e crachás).
- **🛠️ Customização Total:**
  - Cores personalizadas (Fundo e Frente).
  - Controle de tamanho.
  - Margem interna (Padding).
  - **Espaçamento Avançado:** Controle separado para distâncias horizontais e verticais na grade.
- **📊 Múltiplos Formatos:**
  - QR Code
  - Código de Barras (Code 128)
  - Código de Barras (Code 39)
  - EAN-13 (Produtos)
  - UPC (Varejo)
- **👁️ Modos de Visualização:**
  - **Grade:** Para visão geral compacta.
  - **Lista:** Ideal para códigos de barras, com isolamento visual.
- **🌍 Multi-idioma:** Suporte completo para Português (PT), Inglês (EN) e Espanhol (ES).
- **📥 Exportação Fácil:** Baixe seus códigos em PNG com um clique.

## 🛠️ Tecnologias Utilizadas

- **[React](https://reactjs.org/)** - Biblioteca principal.
- **[Vite](https://vitejs.dev/)** - Build tool ultra-rápida.
- **[Framer Motion](https://www.framer.com/motion/)** - Animações complexas e transições suaves.
- **[qrcode.react](https://www.npmjs.com/package/qrcode.react)** - Renderização de QR Codes.
- **[react-barcode](https://www.npmjs.com/package/react-barcode)** - Renderização de Códigos de Barras.
- **[Lucide React](https://lucide.dev/)** - Ícones modernos e leves.
- **CSS Modules / Vanilla CSS** - Estilização performática sem dependência de frameworks pesados.

## 🚀 Como Rodar o Projeto

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/SEU_USUARIO/qr-master.git
    ```
2.  **Instale as dependências:**
    ```bash
    cd qr-master
    npm install
    ```
3.  **Rode o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```
4.  Acesse `http://localhost:5173` no seu navegador.

## 📦 Deploy

Este projeto está configurado para deploy automático no **GitHub Pages** via GitHub Actions. Basta fazer o push para a branch `main` ou `master` que o site será publicado.
